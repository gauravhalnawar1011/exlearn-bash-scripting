ANSI Escape Sequences are used to add color and formatting to the terminal output

printf '%b' '\x61' '\x0a' | xxd 
printf '%b' '\x1b' | xxd
printf '%b' '\x033' | xxd
printf '%b' '\e' | xxd # this is the escape character that is used to start the ANSI escape sequence
printf '\e[1mBold\n' # this will make the text bold
printf '\e[3mItalic\e[0m\n' # this will make the text italic
printf 'not bold  \e[1mbold\e[0m\n' # this will make the text bold and then reset it to normal

printf 'not bold  \e[1mbold\e[0m\n' 

printf 'this is red \e[31mred\e[0m\n' # this will make the text red
printf 'this is yellow \e[33myellow\e[0m\n' # this will make the text yellow
printf 'this is green \e[32mgreen\e[0m\n' # this will make the text green

esc=$'\e' # this is the escape character that is used to start the ANSI escape sequence
printf 'this is blue %b[34mblue%b[0m\n' "$esc"

there is one command called tput that can be used to get the ANSI escape sequences for the terminal capabilities

   printf '\e[38;5;%dm %3d \e[0m' "$i" "$i"
